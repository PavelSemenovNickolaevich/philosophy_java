Задание
1. Измените класс Course следующим образом:
Измените метод getStudents(), чтобы вернуть массив, длина которого совпадает с количеством студентов по дисциплине. (Подсказка: создайте новый массив и скопируйте в него студентов.)
Измените метод addStudent(), чтобы автоматически увеличивать размер массива, если в нем больше нет места для добавления новых студентов. Это делается путем создания нового массива большего размера и копирования в него содержимого текущего массива.
Реализуйте метод dropStudent(), который удаляет указанного студента с дисциплины.
Добавьте новый метод с именем clear(), который удаляет всех студентов с дисциплины.
2. Измените класс TestCourse таким образом, чтобы он создавал дисциплину, добавлял на нее трех студентов, удалял одного из них и отображал имена студентов, обучающихся по этой дисциплине, чтобы за последним студентом не ставилась запятая.
